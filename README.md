# 📝 Blog de Notas CRUD en Python

Este es un proyecto de práctica que implementa un **sistema de notas** con operaciones CRUD (Crear, Leer, Actualizar, Eliminar) usando Python y almacenamiento en archivos JSON.

This is a practice project that implements a **notes system** with CRUD operations (Create, Read, Update, Delete) using Python and JSON file storage.

---

## ⚙️ Funcionalidades / Features
- **[Agregar Nota](ca://s?q=Agregar_nota_en_Python)** → Crea una nueva nota con fecha y hora.  
- **[Ver Notas](ca://s?q=Ver_notas_en_Python)** → Muestra todas las notas guardadas.  
- **[Editar Nota](ca://s?q=Editar_nota_en_Python)** → Permite modificar el contenido de una nota existente.  
- **[Eliminar Nota](ca://s?q=Eliminar_nota_en_Python)** → Borra una nota específica.  
- **[Vaciar Notas](ca://s?q=Vaciar_notas_en_Python)** → Elimina todas las notas del archivo.
